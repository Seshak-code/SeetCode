# SeetCode

SeetCode is a beginner-friendly LeetCode-inspired coding practice platform built with **JavaScript**, **React**, and **Node.js**.

## Stack

- **Frontend:** React + Vite + React Router
- **Backend:** Node.js + Express
- **Package management:** npm workspaces

## Project structure

```text
SeetCode/
  client/      # React app
  server/      # Express API
  shared/      # Shared constants/types (optional expansion)
```

## Getting started

### 1. Install dependencies

```bash
npm install
```

### 2. Start the app

```bash
npm run dev
```

- Frontend: http://localhost:5173
- Backend: http://localhost:5000

## Initial features in this scaffold

- Problem list page
- Problem detail page
- Difficulty badges
- Language selector placeholder
- Starter code panel
- Express API for problems
- Health endpoint
